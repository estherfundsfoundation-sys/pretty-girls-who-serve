# Pretty Girls Who Serve Portal

Static first-release PGWS website built for Vercel deployment.

## What is included

- Animated public homepage
- PGWS Hub preview rooms
- Chapter House and launch pathway preview
- Proverbs 31 Academy interactive module preview
- Resource Vault and membership preview
- Mobile navigation and accessibility basics

## Before launch

Replace the preview buttons with the real:

1. $20 membership checkout URL
2. Group-chat/community URL
3. Chapter application URL
4. Ambassador application URL
5. Social links and business contact email

This version does not accept payments or collect member data yet.
